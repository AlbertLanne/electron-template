# electronuxt-starter
Create an Electron app that serves a Nuxt.js app

## Installation

```bash
npm install # Or yarn install
```

## Development mode

```bash
npm run dev
```

## Production mode

```bash
npm run build
npm run start
```

## Package the app

```bash
npm run pack
```

See more on [electron-packager documentation](https://github.com/electron-userland/electron-packager).
